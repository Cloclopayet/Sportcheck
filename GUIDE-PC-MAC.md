# 💻 SportCheck sur PC et Mac — Guide d'installation

## La solution : installer depuis Chrome ou Edge (2 clics)

Une fois l'app hébergée sur GitHub Pages, vous pouvez l'installer
comme une vraie application sur n'importe quel PC ou Mac —
sans téléchargement, sans installeur, sans droits administrateur.

L'app s'ouvre dans sa propre fenêtre, sans barre d'adresse,
exactement comme une application native.

---

## 🪟 Windows — Chrome ou Edge

### Avec Google Chrome
1. Ouvrez **Chrome** et allez sur votre URL :
   `https://VOTRE-NOM.github.io/sportcheck/`
2. Dans la barre d'adresse, cliquez l'icône **⊕** (ordinateur avec flèche)
   *ou* cliquez les **⋮** (trois points) en haut à droite
3. Cliquez **"Installer SportCheck"** ou **"Installer cette application"**
4. Confirmez → **"Installer"**
5. SportCheck apparaît dans le menu Démarrer et sur le Bureau

### Avec Microsoft Edge (intégré à Windows, recommandé)
1. Ouvrez **Edge** et allez sur votre URL
2. Cliquez les **...** (trois points) en haut à droite
3. **"Applications"** → **"Installer ce site en tant qu'application"**
4. Nom : `SportCheck` → **"Installer"**
5. Une icône SportCheck apparaît sur le Bureau et dans la barre des tâches

### Raccourci Bureau (alternative rapide)
Si vous ne voyez pas l'option d'installation :
- Chrome : Menu ⋮ → **"Plus d'outils"** → **"Créer un raccourci"**
  → Cochez **"Ouvrir dans une fenêtre"** → **"Créer"**

---

## 🍎 Mac — Chrome ou Safari

### Avec Google Chrome
1. Ouvrez **Chrome** et allez sur votre URL
2. Dans la barre d'adresse, cliquez l'icône **⊕**
   *ou* Menu **⋮** → **"Installer SportCheck"**
3. Cliquez **"Installer"**
4. SportCheck apparaît dans votre **Launchpad** et **Dock**

### Avec Safari (macOS Sonoma 14+ uniquement)
1. Ouvrez **Safari** et allez sur votre URL
2. Menu **Fichier** → **"Ajouter au Dock..."**
3. Nom : `SportCheck` → **"Ajouter"**
4. L'app apparaît dans le Dock comme une application native

---

## ✅ Résultat final — ce que vous obtenez

Après installation, SportCheck :
- S'ouvre dans **sa propre fenêtre** (sans barre d'adresse)
- Apparaît dans la **barre des tâches** (Windows) ou le **Dock** (Mac)
- Est accessible depuis le **menu Démarrer** ou **Launchpad**
- Fonctionne **hors ligne** pour saisir les rapports
- Se **synchronise automatiquement** quand internet est disponible
- Se **met à jour automatiquement** dès que vous mettez à jour GitHub

## ❓ FAQ

**Peut-on utiliser sans internet ?**
Oui — la saisie, la grille de contrôle et la génération PDF
fonctionnent hors ligne. Seuls l'envoi Gmail et la sauvegarde
Drive nécessitent une connexion.

**Les données sont-elles partagées entre les appareils ?**
Non — chaque appareil garde ses propres données locales.
Les rapports finalisés sont envoyés par Gmail et sauvegardés
sur Drive, ce qui les rend accessibles partout.

**Peut-on installer sur plusieurs PC ?**
Oui — autant d'installations que nécessaire, sur la même URL.
Chaque technicien installe sur son propre appareil.

**Comment désinstaller ?**
- Windows : Menu Démarrer → clic droit sur SportCheck → Désinstaller
- Mac : Launchpad → maintenir l'icône → cliquer la croix
