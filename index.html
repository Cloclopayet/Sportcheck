📧 Gmail — Envoi direct

Envoi depuis votre compte Gmail directement.
1. Allez sur console.cloud.google.com
2. Créez un projet → activez Gmail API
3. Identifiants → OAuth 2.0 → Type : Application Web
4. Ajoutez l'URL de cette app dans "Origines JavaScript autorisées"
5. Collez le Client ID ci-dessous — c'est tout !
⚠️ Même Client ID que Google Drive si déjà configuré.
Le rapport sera envoyé depuis adv.gsemu@gmail.com vers cette adresse.

☁️ Google Drive (optionnel)

Upload automatique dans votre Drive.
1. console.cloud.google.com → Nouveau projet → Activer Drive API
2. Identifiants → OAuth 2.0 → Application Web
3. Ajouter l'URL de ce fichier dans "Origines autorisées"
Sur Drive : clic droit sur le dossier → Obtenir le lien
L'ID est la partie après folders/ dans l'URL
⚡ Mode fichier local détecté
Gmail API et Google Drive nécessitent HTTPS.
En mode local : Gmail → ouvre votre client mail (Outlook, Mail, etc.)
Drive → sauvegarde un fichier HTML sur votre PC.
Pour toutes les fonctions : hébergez sur GitHub Pages.
